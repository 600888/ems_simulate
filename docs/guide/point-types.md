# 测点类型

EMS Simulate 支持四种测点类型，覆盖工业自动化中常见的数据采集和控制需求。

## 类型概览

| 类型 | 代码 | 说明 | 典型用途 |
|------|------|------|----------|
| **遥测 (YC)** | frame_type=0 | 模拟量测量 | 电压、电流、功率、温度 |
| **遥信 (YX)** | frame_type=1 | 开关量状态 | 运行状态、故障标志 |
| **遥控 (YK)** | frame_type=2 | 开关量命令 | 启停命令、合分闸 |
| **遥调 (YT)** | frame_type=3 | 模拟量命令 | 功率设定、温度设定 |

## 遥测 (YC - Telemetering)

用于采集模拟量测量值，如电压、电流、功率、温度等连续变化的物理量。

**特点：**
- 支持浮点数和整数类型
- 可配置乘法系数和加法系数
- 支持多种数据解析格式

**系数转换公式：**
```
真实值 = 寄存器值 × 乘法系数 + 加法系数
寄存器值 = (真实值 - 加法系数) ÷ 乘法系数
```

## 遥信 (YX - Status Indication)

用于采集开关量状态，如设备运行/停止、开关分/合、故障/正常等二值状态。

**特点：**
- 值通常为 0 或 1
- 可配置反转逻辑
- 支持状态变化事件

## 遥控 (YK - Remote Control)

用于发送开关量控制命令，如启动/停止设备、合闸/分闸等。

**特点：**
- 支持"选择-执行"两步操作
- 可配置命令超时
- 支持返校验证

## 遥调 (YT - Setpoint)

用于发送模拟量设定值，如功率设定、温度设定等。

**特点：**
- 支持范围校验
- 支持步进值限制
- 可配置系数转换

## 数据模拟方式

每种测点都支持多种数据模拟方式：

| 模拟方式 | 说明 |
|----------|------|
| **固定值** | 保持固定数值不变 |
| **随机模拟** | 在指定范围内随机变化 |
| **步进模拟** | 按指定步长递增/递减 |
| **正弦模拟** | 按正弦曲线变化 |
